# SmartCrosshair v2.0.0

**A resource-pack compatible, fully customizable PvP crosshair mod for Fabric Minecraft.**

---

## **New in v2.0.0**

- **Complete in-game config menu with ClothConfig and ModMenu integration**
- **Profile management:** save, switch, and instantly apply unlimited custom crosshair profiles
- **Per-entity color and toggle:** Set crosshair color and enable/disable for Players, Hostile Mobs, Passive Mobs, Armor Stands, and Villagers
- **Outline color and thickness, alpha (transparency), size, FOV scaling, smoothness settings**
- **Lightweight mode, color blind mode, "Only Show On Target," transparent GUI, disable in spectator, PvP/minimal/classic presets**
- **Automatic update checker and version popup when mod loads**
- **No resource pack override, fully compatible with all custom crosshair textures**
- **Keybind for toggling mod, and all settings respected live**

---

**Core Features**
- **Dynamic tint:** Crosshair changes color for each entity type you aim at—players, mobs, animals, armor stands, villagers, etc.
- **Dramatic, customizable coloring:**  
  - Set your own RGB color for every entity group
  - Classic, PvP, and Minimal presets included  
- **All-in-one settings menu:** Every feature configurable in Minecraft GUI
- **Quick profile switching:** Instantly swap between saved configs for PvP, chill, or custom play
- **Server-safe, client-only** and highly optimized
- **For Minecraft 1.21 (Fabric)**

---

## **How to Use**

1. Put the SmartCrosshair jar into your `.minecraft/mods` folder (with Fabric Loader).
2. Start Minecraft and open the config from Mod Menu or via the config keybind.
3. Set and save your colors, toggles, size, and other options.
4. (Optional) Save custom profiles or switch between them on the fly!
5. Use your custom crosshair with any resource pack!
