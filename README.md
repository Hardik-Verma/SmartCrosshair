# SmartCrosshair

**A resource-pack compatible, PvP-focused crosshair mod for Fabric Minecraft.**  
Created and maintained by _Pheonix

---

**Features**
- **Dynamic tint:** Crosshair turns red when aiming at any entity (players, mobs, armor stands, animals, etc.)
- **Dramatic distance-based coloring:**  
  - Very close: maroon/dark red  
  - Close: classic bright red  
  - Far: faded light red  
- **Works in both first and third person**
- **Fully resource-pack compatible:** Doesn't override crosshair textures
- **Keybind to toggle crosshair effect on/off**
- **Lightweight, client-side only, server-safe**
- **Automatic update checker with popup**
- **Supports Minecraft 1.20.5 and 1.21**

---

## How to Use

1. Drop the latest jar into your `.minecraft/mods` folder (Fabric loader required).
2. Start Minecraft. Crosshair will react to any entity you aim at.
3. Set your toggle keybind in Options → Controls → SmartCrosshair section.
